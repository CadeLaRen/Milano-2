Milano
--------
##An android cookies manager library

Download
--------

Grab via Gradle:
```groovy
compile 'com.intrusoft.milano:milano:1.0.0'
```

Features
--------
- Automatic Mangage Cookies.
- Customizable

Implementation
-----------
###Initially:
- manageCookies method set to false.
- Request method is set to "GET"

#### To make GET request

```java
Miliano.along(context)
    .
    
```
